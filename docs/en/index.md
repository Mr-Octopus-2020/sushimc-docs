# SushiMC Player Guide

Welcome to **SushiMC** — a Minecraft network server!

> 🚧 This guide is a work in progress. If you have questions, join our QQ group.

---

## Game Modes

SushiMC is a Velocity proxy network running multiple modpacks. Pick your adventure below!

| Game Mode | Description |
|-----------|-------------|
| [**Zenith**](zenith.md) | A laid-back survival, exploration & building modpack — now in open beta. [**Enter guide →**](zenith.md) |
| **Myth(WIP)** | With the theme of agriculture, magic, industry. Released in 2025, currently under maintainance. |
| **New Wardon(WIP)** | Creative building. Currently not public, and under maintainance. |

!!! tip "New here?"

    First time playing? Currently we have only one modpack - Join [**Zenith**](zenith.md#guide-join) to get started in just 2 steps!
