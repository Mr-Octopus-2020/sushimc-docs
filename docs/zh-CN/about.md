# 关于寿司峰

## 我们是寿司！

> 寿司峰是一座山峰，但它也可能是寿司。对，我们是寿司，对吗？我们也可能是一个服务器，不对……

寿司峰是一个始于2026年的 Minecraft 服务器项目，目前由寿司峰开发团队维护。
这是一个业余项目，我们希望能给玩家们提供优质的 Minecraft 游戏服务。

……

算了我写不动了。请您**自行想象**寿司峰的过去，现在和未来！

!!! note "你知道吗？"
    “寿司峰”这个名字是意外诞生的。一位管理员曾在建议新名字时，将管理组QQ群命名为“寿司峰端管理组”，后该群名被其他管理“错误地改正”，变成了今天的寿司峰。
